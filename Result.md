# Отчет о результатах автоматизированного тестирования
## Мобильное приложение «Мобильный хоспис»

---
## 1. Сравнение ручного и автоматизированного тестирования

| Критерий                   | Ручное тестирование     | Автоматизированное тестирование |
|----------------------------|-------------------------|---------------------------------|
| Время выполнения 54 тестов | ~3 часа                 | ~10 минут                       |
| Стабильность результатов   | Зависит от человеческого фактора | Высокая                         |
| Отчетность                 | Ручное оформление       | Автоматическая (Allure)         |

---

## 2. Техническая реализация

### 2.1. Используемый технологический стек

- **Язык программирования:** Java 8
- **Фреймворк тестирования:** Espresso 3.4.0
- **Организация тестов:** JUnit 4
- **Паттерн проектирования:** Page Object Model (POM)
- **Генерация отчетов:** Allure Framework 2.4.0
- **Система сборки:** Gradle 7.2

### 2.2. Архитектура тестов

Тесты организованы по модульному принципу:

| Модуль | Описание | Количество классов | Классы |
|--------|----------|-------------------|--------|
| `tests/` | Тестовые классы | 5 | `AboutTests`, `LoginTests`, `MainTests`, `NewsTests`, `QuotesTests` |
| `pages/` | Page Objects | 7 | `AboutPage`, `CreateEditNewsPage`, `FilterNewsPage`, `LoginPage`, `MainPage`, `NewsPage`, `QuotesPage` |
| `utils/` | Утилиты | 1 | `WaitAction` |
